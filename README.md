# FUTURE_CS_01: Vulnerability Assessment Report

## 1. Website Tested
The target for this assessment is a controlled environment provided by Acunetix for security testing purposes:
* **Target URL:** http://testaspnet.vulnweb.com
* **Infrastructure:** Windows Server running Microsoft IIS 8.5

## 2. Scope
The scope of this task is a **Black-Box Vulnerability Assessment**. This includes:
* **Passive Reconnaissance:** Identifying server versions, headers, and technological stack.
* **Network Mapping:** Identifying open ports and active services.
* **Vulnerability Scanning:** Detecting security weaknesses, misconfigurations, and outdated software versions.
* **Reporting:** Documenting findings with risk levels and recommended fixes.
* All scanning was performed passively to comply with ethical guidelines and ensure zero impact on target availability

## 3. Tools Used
To ensure a comprehensive assessment, the following industry-standard tools were utilized:
* **Nmap:** For network discovery, port scanning, and service version detection.
* **Browser DevTools:** For manual inspection of HTTP response headers and security flags.
* **OWASP ZAP:** For automated web application security scanning and vulnerability identification.
* **Canva:** For the professional design and layout of the final Vulnerability Assessment Report.

## 4. Repository Structure
* **/Evidence**: Contains all raw tool outputs and screenshots.
* **/Report**: Contains the final PDF report designed in Canva.
